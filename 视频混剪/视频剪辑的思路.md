# 一、需求分析
首先提出几个问题：

1. 我要做什么类型的视频？是影视混剪？是电视剧剪辑？是搬运视频？
2. 市场上有哪些视频类型？这些视频的播放量如何？
3. 确定好要做的视频类型后，需要哪些工具来做？以及这些工具的效果如何？
4. 收益分析。收益从哪里获得？比例如何？
5. 发布平台有哪些？
6. 发布视频有哪些规范要求？
